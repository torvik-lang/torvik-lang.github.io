# Contact

Questions, ideas, bug reports, or just want to say hello? Any of these reach us directly.

## Email

[torviklang@gmail.com](mailto:torviklang@gmail.com)

Best for anything detailed — build-tooling requests, security reports, or longer
questions.

## Social

- **X** — [@Torviklang](https://x.com/Torviklang)
- **Instagram** — [@torviklang](https://instagram.com/torviklang)

Release announcements go out on X first.

## On GitHub

For bugs and feature requests, an issue on the relevant repository is often the fastest
path — see the [contributing page](/contributing) for links to each project's tracker.
Prefer to talk things through? The
[Discussions board](https://github.com/torvik-lang/torvik/discussions) is open for
questions, ideas, and show-and-tell.
